# GitHub MCP Custom (Cross-Platform NPX Wrapper)\n\nA custom-packaged version of the [GitHub MCP Server](https://github.com/github/github-mcp-server) that runs seamlessly on **Windows**, **Linux**, and **macOS** via `npx`. This wraps the Go binary in a Node.js launcher so developers using Claude Desktop can use GitHub MCP capabilities with no Docker or manual compilation required.\n\n---\n\n## 📦 Why This Exists\n\nThe official GitHub MCP Server requires:\n- Docker or\n- Manual Go compilation.\n\nThese options work well on Linux/macOS, but **break or complicate Windows usage**.\n\nThis package makes the server available via `npx`, eliminating the need for:\n- Docker\n- Go installation\n- Custom PATH setups\n\n---\n\n## 🧠 What This Package Does\n\n- Provides a cross-platform `github-mcp-custom` command\n- Executes the prebuilt Go binary via Node.js wrapper\n- Fully NPX-compatible on all OSes\n- Works out-of-the-box with Claude Desktop or other MCP-compatible tools\n- **Protocol version compatible with Claude Desktop**\n\n---\n\n## 🚀 Installation & Usage\n\n### 🔹 Option 1: Run with NPX (Recommended)\n\n```bash\nnpx -y github-mcp-custom@1.0.20 stdio\n```\n\n> First-time use will download the package and run it automatically.\n\n### 🔹 Option 2: Install Globally\n\n```bash\nnpm install -g github-mcp-custom@1.0.20\ngithub-mcp-custom stdio\n```\n\n### 🔹 Option 3: Direct Executable (Windows Fallback)\n\nIf NPX has issues on Windows, use the direct path:\n\n```json\n{\n  \"mcpServers\": {\n    \"github\": {\n      \"command\": \"C:\\\\Users\\\\[USERNAME]\\\\AppData\\\\Roaming\\\\npm\\\\node_modules\\\\github-mcp-custom\\\\dist\\\\github-mcp-server.exe\",\n      \"args\": [\"stdio\"],\n      \"env\": {\n        \"GITHUB_PERSONAL_ACCESS_TOKEN\": \"your_token_here\"\n      }\n    }\n  }\n}\n```\n\nReplace `[USERNAME]` with your actual Windows username.\n\n**Note:** Use `@1.0.20` or later for optimal Claude Desktop compatibility.\n\n---\n\n## 🛠️ Building from Source\n\n### Prerequisites\n\n- [Go](https://golang.org/dl/) 1.23.7 or later\n- [Node.js](https://nodejs.org/) v14 or later\n- Git\n\n### Clone and Build\n\n```bash\n# Clone the repository\ngit clone https://github.com/idletoaster/github-mcp-custom.git\ncd github-mcp-custom\n\n# Install dependencies\ngo mod tidy\n\n# Build binaries for all platforms\nnpm run build\n\n# Or build for specific platforms:\nnpm run build:linux    # Linux binary\nnpm run build:macos    # macOS binary  \nnpm run build:win      # Windows binary\n\n# Test the build\nnode bin/index.js stdio\n```\n\n### Package for NPM\n\n```bash\n# Create NPM package\nnpm pack\n\n# Publish to NPM (requires npm login)\nnpm publish\n```\n\n### Manual Build Commands\n\n```bash\n# Linux\nGOOS=linux GOARCH=amd64 go build -ldflags \"-s -w\" -o dist/github-mcp-server-linux .\n\n# macOS\nGOOS=darwin GOARCH=amd64 go build -ldflags \"-s -w\" -o dist/github-mcp-server-macos .\n\n# Windows\nGOOS=windows GOARCH=amd64 go build -ldflags \"-s -w\" -o dist/github-mcp-server.exe .\n```\n\n---\n\n## 🛠️ Prerequisites\n\n- [Node.js](https://nodejs.org) v14 or later must be installed\n- An active GitHub Personal Access Token with proper scopes:\n  - `repo` (for repository access)\n  - `read:org` (for organization access)\n  - `user` (for user information)\n\n---\n\n## 🔐 Creating a GitHub Personal Access Token\n\n1. Go to [GitHub Settings > Developer Settings > Personal Access Tokens](https://github.com/settings/tokens)\n2. Click \"Generate new token (classic)\"\n3. Select these scopes:\n   - ✅ `repo` - Full control of private repositories\n   - ✅ `read:org` - Read org and team membership\n   - ✅ `user` - Update user data\n4. Copy the generated token\n\n---\n\n## 🖥️ Claude Desktop Configuration\n\n### Windows\n\nAdd this to your Claude Desktop configuration file located GitHub MCP tools:\n\n### 📂 **Repository Management**\n- `search_repositories` - Search GitHub repositories\n- `get_file_contents` - Read file contents from repos\n- `create_or_update_file` - Create/modify files\n- `create_repository` - Create new repositories\n- `fork_repository` - Fork repositories\n- `create_branch` - Create branches\n- `push_files` - Push file changes\n- `list_commits` - View commit history\n- `get_commit` - Get specific commit details\n\n### 🐛 **Issues & Project Management**\n- `get_issue` - Get issue details with timestamps\n- `create_issue` - Create new issues\n- `update_issue` - Update issues (returns updated info with timestamps)\n- `search_issues` - Search across issues\n- `list_issues` - List repository issues\n- `get_issue_comments` - Get issue comments with timestamps\n- `add_issue_comment` - Add comments to issues\n\n### 🔄 **Pull Requests**\n- `get_pull_request` - Get PR details\n- `create_pull_request` - Create new PRs\n- `update_pull_request` - Update existing PRs\n- `merge_pull_request` - Merge PRs\n- `get_pull_request_files` - See PR changes\n- `get_pull_request_comments` - Get PR comments\n- `create_and_submit_pull_request_review` - Review PRs\n\n### 🔒 **Security & Scanning**\n- `list_code_scanning_alerts` - Code security alerts\n- `list_secret_scanning_alerts` - Secret scanning alerts\n\n### 🔔 **Notifications**\n- `list_notifications` - Get notifications\n- `dismiss_notification` - Dismiss notifications\n- `mark_all_notifications_read` - Mark all as read\n\n### 👤 **User & Context**\n- `get_me` - Get authenticated user details\n- `search_users` - Search GitHub users\n\n---\n\n## 🧪 Testing Your Setup\n\n1. **Test the command directly:**\n   ```bash\n   npx -y github-mcp-custom@1.0.20 stdio\n   ```\n   Should output: `GitHub MCP Server running on stdio`\n\n2. **Test with Claude Desktop:**\n   - Restart Claude Desktop after updating the config\n   - Try asking: \"Can you get my GitHub user information?\"\n   - Claude should be able to use the `get_me` tool\n\n---\n\n## 🔧 Troubleshooting\n\n### Common Issues\n\n**❌ \"Server disconnected\" in Claude Desktop**\n- Ensure you're using version `1.0.20` or later\n- Check that your GitHub token is valid and has proper scopes\n- Verify the JSON configuration is valid (no trailing commas)\n\n**❌ \"Command not found\"**\n- Make sure Node.js is installed and in your PATH\n- Try running `npm --version` to verify npm is working\n\n**❌ \"Authentication failed\"**\n- Verify your GitHub Personal Access Token is correct\n- Check that the token has the required scopes (`repo`, `read:org`, `user`)\n- Make sure the token hasn't expired\n\n**❌ \"Method not found\" errors**\n- Update to the latest version: `npm install -g github-mcp-custom@latest`\n\n**❌ \"Binary not found\" during build**\n- Run `npm run build` to create the required binaries\n- Ensure Go is installed and in your PATH\n- Check that the `dist/` directory exists and contains binaries\n\n---\n\n## 📁 Project Structure\n\n```\ngithub-mcp-custom/\n├── bin/\n│   └── index.js              # Node.js wrapper (NPM entry point)\n├── dist/
